# TCC Bridge V2.1.0 - "Kael Edition" (NEW VERSION)

## One-Tap Installation
Run this in Termux:
```bash
curl -sS https://raw.githubusercontent.com/Dzongy/tcc-bridge/main/deploy_v2.1.sh | bash
```

## Files
- `bridge_v2.1.py`: Master Bridge Script (Port 8080)
- `ecosystem_v2.1.js`: PM2 Config
- `boot_v2.1.sh`: Boot Script
- `deploy_v2.1.sh`: Installer
